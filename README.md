# gridlock.css

Small CSS grid layout implemented with flexbox.

<br>

## Installation

```bash
npm install --save gridlock.css
```

## Usage

```html
<div class="Grid">
  <div class="Row">
    <div class="Cell">
    </div>
  </div>
</div>
```

Look at modifiers [here](https://github.com/caiogondim/gridlock.css/blob/master/src/index.css).
